Smart Complaint Management System :-
This is a web application that allows users to submit complaints and track their status, while the admin can manage and resolve them. 
The goal of this project is to provide a simple digital system for complaint handling instead of manual paperwork.

What the system does:-
Users can:
•	Submit complaints
•	Receive automatic email confirmation
•	Track progress
•	View complaint history
Admin can:
•	View all complaints
•	Update status (Pending / In Progress / Resolved)
•	Delete complaints
Entire system fully developed and implemented by me

Workflow :-
•	User submits complaint
•	Data stored in MySQL
